**Opening Times App**

This is an application that informs the user of a shop's opening times, includng whether the shop is currently open or closed.

This is done using PHP and JavaScript. The time is monitored consistently so the page can update when it reaches a preset opening or closing time.

Although this is a relatively simple demonstration, the concept and functionality can be applied to other more complex applications, such as a GP Booking System, for example, that only allows booking appointments during certain times of the day/week.
